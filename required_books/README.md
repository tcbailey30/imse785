# Required Books for IMSE 785 - Big Data Analytics:                             
1. Bengfort, B. and Kim, J. (2016) Data Analytics with Hadoop, 1st Ed. O’Reilly.
2. Leskovec, J., Rajaraman, A. and J. D. Ullman (2020) Mining of Massive Datasets, 3rd ed., Cambridge,  NY. (Note that 2nd ed will also work.)
3. Chambers, B. and Zaharia, M. (2018) Spark the Definitive Guide. 1st Ed. O’Reilly
4. White, T. (2012) Hadoop: The Definitive Guide, 3rd. O’Reilly
5. Ryza, S., Laserson, U., Owen, S., and Wills, J. (2017). Advanced Analytics with Spark, 2nd ed., O’Reilly.
6. McKinney, W. (2013) Python for Data Analysis, O’Reilly.
7. Muller, A. C. and Guido, S. (2017) Introduction to Machine Learning with Python: A Guide for Data Scientist, 1st Ed., O’Reilly.
